## How to use
Make sure you in the folder has file like this:
```
gothonweb/
├── bin
├── docs
├── ex47
├── gothonweb
├── sessions
├── static
├── templates
└── tests
```
Then input this in your shell:
```bash
export PYTHONPATH=$PYTHONPATH:.
python /bin/gameapp.py
```
Input http://localhost:8080/ in your browser, Start Game.
